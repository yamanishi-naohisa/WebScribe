# 変更履歴

## [未リリース] - 2025-12-07

### 追加機能
- **ログイン機能の実装**
  - ログインが必要なページに対応
  - ログインURL、ユーザー名、パスワードの入力に対応
  - セレクタの自動検出機能（ユーザー名、パスワード、ログインボタン）
  - ログイン後に目的のページに自動遷移して要素を取得
  - GUIアプリケーションにログイン情報入力欄を追加

### 修正
- **Reactアプリケーション対応の改善**
  - JavaScriptで動的に生成されるコンテンツが正しく取得できない問題を修正
  - 実際に表示されているコンテンツと取得データが異なる問題を解決

### 追加機能（以前）
- **JavaScript実行完了の待機機能**
  - `document.readyState`を監視してページの読み込み完了を確認
  - ページの高さが安定するまで待機してから要素を取得
  - スクロール処理で遅延読み込みコンテンツを読み込む

- **自動化検出の回避強化**
  - より完全なユーザーエージェント設定
  - CDPコマンドによる自動化検出の回避
  - JavaScriptによるnavigator.webdriverプロパティの隠蔽

- **ブラウザ設定の改善**
  - 新しいヘッドレスモード（`--headless=new`）の使用
  - ウィンドウサイズの設定（モバイルサイト対策）
  - 自動化検出を回避するための追加オプション

### 改善
- **ページ読み込み処理の最適化**
  - スクロール処理でページ全体を読み込んでから要素を取得
  - ページの変更がないことを確認してから要素収集を開始
  - より適切な待機時間の設定

- GUIアプリケーション
